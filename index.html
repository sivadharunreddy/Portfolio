Automatic Plant Irrigation System

The Automatic Plant Irrigation System uses an Arduino with a soil moisture sensor to monitor soil dryness. When moisture drops below a threshold, a water pump is triggered via relay to irrigate the plant. This project ensures efficient water usage and is ideal for smart gardening applications.

Technologies Used: Arduino Uno, Soil Moisture Sensor, Relay Module, Water Pump, C++ (Arduino IDE), Python, Breadboard & Jumper Wires

Speech to Text using Deep Learning

This project converts spoken language into written text using deep learning techniques. It uses Mel-frequency cepstral coefficients (MFCCs) for audio feature extraction and a Recurrent Neural Network (RNN) with LSTM layers for sequence modeling. The model is trained on audio datasets to accurately transcribe human speech into readable text.

Technologies Used: Python, Deep Learning, TensorFlow/Keras, NumPy, Librosa, RNN, LSTM, Speech Recognition, Jupyter Notebook.
